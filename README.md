# AI Writing Assistant

AI Writing Assistant is a modern web application that helps users improve their writing by detecting spelling mistakes and grammar issues using AI-powered text analysis. The application provides real-time feedback through a clean, responsive, and user-friendly interface.

## Features

* AI-powered grammar checking
* Spelling mistake detection and correction suggestions
* Real-time text analysis
* Responsive and modern user interface
* Fast and efficient performance
* Easy-to-use writing enhancement tools

## Technologies Used

### Frontend

* React.js
* JavaScript
* Tailwind CSS

### Backend

* Node.js
* Express.js

## Installation

### Clone the Repository

```bash
git clone https://github.com/Parth-js/AI-Writing-Assistant.git
cd ai-writing-assistant
```

### Install Frontend Dependencies

```bash
cd client
npm install
```

### Install Backend Dependencies

```bash
cd ../server
npm install
```

## Running the Application

### Start Backend Server

```bash
cd server
npm start
```

### Start Frontend Application

```bash
cd client
npm start
```

The frontend and backend servers will run locally, allowing users to access AI-powered writing assistance through the browser.
